Install（参照[Laravel 5 系列入门教程（一）](http://lvwenhan.com/laravel/432.html)）
====
* 安装php，开启mcrypt扩展
* 安装composer包管理器
```bash
curl -sS https://getcomposer.org/installer | php
```
* 使用命令行安装laravel 5
```bash
composer create-project laravel/laravel learnlaravel5
```
